Welcome to G's World!

G's World is a small videogame initially made for a project, and now published publically for anyone to explore and experience this short interactive story.
Know more about it here.

Tips

  • Creating an account is mandatory to play the game, there is no data being collected and the password is encoded before being stored in the database. However, there is no way to change your password or retrieve a user name so don't forget it.
  • Press 'esc' and the "Save" button to save your progress and to ensure that you don't have to start over if you die or get disconnected.
  • Remember this game is a project made by one person so some details are not as elaborate as other bigger games, but if you encounter a bug, don't hesitate to get in contact with me.
