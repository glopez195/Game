About

G's World is a Final Proyect I am currently working on for the CS50X course. But beyond that it has become a personal project and a goal I am commited to finish and polish until I am satisfy with the results, perhaps never. The projects consists of a little game that takes place in the world I've created (G). All the coding is done in Vanilla JavaScript with the help of some API's, HTML, CSS and the work of some awsome artists in Itch.io (don't forget to check all their work in Credits).
The game is under development so you wont find much to do for now but if you follow up with it you might get surprised soon.
In order to create a great user experience I have created a Backend part for this project using Python with Flask and SQLite3 for saving the data in the server, there is no need to find a save button with just closing the window the game will automatically save your progress, which at the moment is just the position you are in the map.
I really hope this project finds some people that would be interest in exploring it and get to enjoy this piece of me on the vast WWW.
