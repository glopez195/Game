About

G's World is a project I started working on for the CS50X course. However, for me, it has evolved to become a personal project and goal I am committed to completing and polishing until I am satisfied with the results (perhaps never). The project consists of a game that takes place in the world I've created (G). All the coding is done in Vanilla JavaScript with the help of some API's, HTML, CSS and the work of some awesome artists in Itch.io (don't forget to check out their work in Credits).
The game is still under development so there's not much to do for now, but if you come back soon you might be surprised.
In order to create a great user experience, I have designed a Backend part for this project using Python with Flask and SQLite3 for saving the data in the server. There is no need to press a save button, by just closing the window the game will automatically save your progress.
I hope this project finds people who are interested in exploring G's World and that they get to enjoy this piece of me on the vast WWW.
